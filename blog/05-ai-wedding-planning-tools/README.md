# AI wedding planning (timelines, budgets, vendor emails)

## Purpose of this brief
This README is an internal “blog post brief” for a practical guide to using AI tools during wedding planning, with guardrails so it helps without making things generic.

## Why this topic is trending (signals to reference)
- **Major adoption**: Zola reports 54% of couples use AI in some way to plan their wedding, with rapid year-over-year growth.
  - Source: https://www.zola.com/expert-advice/the-first-look-report-2026
- **What couples use AI for most**: Zola lists etiquette questions (54%), timelines/to-do lists (44%), drafting emails (40%), decor inspiration (31%), budgets (27%), travel (27%).
  - Source: https://www.zola.com/expert-advice/the-first-look-report-2026
- **AI-assisted writing is a named wedding tech trend**: The Knot highlights using AI to assist writing (with the caveat to keep it personal).
  - Source: https://www.theknot.com/content/wedding-technology-trends

## Audience + intent
- **Primary reader**
  - Couples overwhelmed by planning tasks and decision fatigue.
  - Couples who want templates for emails/timelines/budgets.
- **Search intent**
  - “AI wedding planning”
  - “wedding timeline generator”
  - “wedding vendor email template”
  - “AI wedding budget”

## Angle / positioning
Make AI feel like a helpful assistant: fast drafts + better organization + fewer awkward emails. Keep “meaning” work (vows, big choices) human.
